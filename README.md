# Godot STG模板

#### 介绍
一个简单的Godot STG模板，方便Godot用户用于制作类东方的弹幕游戏。

#### 更新说明
近期将专注于其它项目，可能不太有多少时间更新。
### Alpha-1.0
完成了基础的弹幕制作功能，
完成了基础的BOSS，自机，敌机，符卡设定。
完成了基础的数值属性：生命，炸弹，得分，得点，火力，擦弹设定。
完成了简易的计分系统。
性能和易用性方面有待优化和完善，内容方面也有待充实。

### Alpha-1.1
提供了对godot4.0rc1的支持
优化和修正了弹幕机制与配置
添加了弹幕标签功能，可给弹幕添加tags
完善了激光发射的功能
